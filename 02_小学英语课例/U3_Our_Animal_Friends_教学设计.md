# U3 Our Animal Friends 教学设计

## 一、基本信息

| 项目     | 内容                  |
| -------- | --------------------- |
| 学段     | 小学三年级上册        |
| 单元     | U3 Our Animal Friends |
| 课型     | 对话课 / 描述表达课   |
| 课时     | 1课时，约40分钟       |
| 主题语境 | 动物朋友与特征描述    |
| 设计者   | 曹月华                |

## 二、教材与学情分析

本单元以“动物朋友”为主题，契合三年级学生兴趣特点。动物图片、声音竞猜、动作模仿和小组展示都能有效调动学生参与。三年级学生已具备初步听说模仿能力，但完整描述能力仍需句型支架。

本课设计重点是引导学生从识别动物词汇走向描述动物特征，并通过问题链培养观察、比较和简单推测能力。

## 三、教学目标

### 语言能力

- 能识别并说出常见动物词汇；
- 能使用简单形容词描述动物颜色、大小或能力；
- 能用目标句型介绍一个 animal friend。

### 思维品质

- 能观察动物外形特征；
- 能比较不同动物的相同点和不同点；
- 能根据声音、动作或局部图片推测动物名称。

### 学习能力

- 能在小组合作中完成动物介绍任务；
- 能借助句型支架进行有序表达。

### 文化意识

- 形成爱护动物、尊重生命的初步意识；
- 理解人与动物可以友好相处。

## 四、教学重难点

### 教学重点

- 动物词汇与特征描述；
- 使用目标句型进行动物介绍。

### 教学难点

- 学生能从单词输出过渡到完整句表达；
- 学生能在小组任务中完成合作表达。

## 五、教学方法

- 情境教学法；
- TPR 活动；
- 图片预测；
- 分层提问；
- 小组合作展示。

## 六、教学过程

### Step 1 Warm-up and Guessing Game

教师播放动物声音或展示局部图片，提问：

- What is it?
- Is it big or small?
- What color is it?

设计意图：激发兴趣，导入 animal friend 主题。

### Step 2 Presentation

教师通过图片呈现动物词汇和描述性语言。

目标句型示例：

```text
It is a ______.
It is ______.
It can ______.
I like it.
```

设计意图：帮助学生建立“动物名称—外形特征—能力表达”的语言链条。

### Step 3 Practice

活动一：Listen and Point  
学生听动物名称或声音，指出相应图片。

活动二：Act and Guess  
一名学生模仿动物动作，其他学生猜测。

活动三：Ask and Answer  
两人一组完成问答。

```text
A: What is it?
B: It is a ______.
A: What color is it?
B: It is ______.
```

### Step 4 Production

小组完成 `Our Animal Friend` 展示任务。

输出示例：

```text
It is a cat.
It is white.
It can run.
I like it.
```

分层要求：

- 基础层：能说动物名称；
- 提升层：能说颜色或大小；
- 拓展层：能说能力或喜爱原因。

### Step 5 Summary and Evaluation

教师引导学生总结：

```text
Animals are our friends. We should love animals.
```

评价维度：

- 词汇准确；
- 表达完整；
- 小组合作；
- 愿意展示。

## 八、作业设计

1. 完成 `Our Animal Friend` 活动单；
2. 向家人介绍一种动物；
3. 选做：画一只动物并写 3 句英文介绍。

## 九、教学反思

动物主题容易激发学生兴趣，但课堂中要避免只停留在热闹的游戏层面。教师需要通过问题链引导学生观察、比较和表达，让游戏服务于语言目标。后续可加入更多开放性问题，如“Why do you like it?”，帮助学生从描述走向个性化表达。
