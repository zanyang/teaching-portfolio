学情诊断与家校反馈模板

一、使用场景

本模板用于展示教师如何根据课堂表现、作业情况、错题类型和阶段测评结果，对学生进行学习诊断，并向家长提供具体、客观、可执行的反馈。

适用于：

  • 初中英语阶段反馈；
  • 小学英语学习习惯反馈；
  • 课后服务和个性化辅导；
  • 家校沟通记录。

二、学生阶段反馈样例（匿名）

学生基本情况

学生：A同学

年级：初二

阶段：阅读理解专项训练后

主要表现：词汇基础尚可，但阅读细节题失分较多，写作句式较单一。

三、学情诊断

| 观察维度 | 表现 | 初步判断 |

| -------- | -------------------------------- | -------------------- |

| 词汇 | 高频词掌握较好，但长难词影响理解 | 需要积累话题词汇 |

| 阅读 | 细节题失分较多 | 关键词定位能力不足 |

| 语法 | 能识别基础结构，复杂句分析不稳定 | 长句拆分训练不足 |

| 写作 | 能写基础句，但连接词少 | 句式储备不足 |

| 学习习惯 | 愿意订正，但错因记录不完整 | 需要建立错题复盘流程 |

四、阶段提升建议

  1. 阅读训练中先圈出题干关键词，再回原文定位。
  2. 每周完成 2 篇阅读错题复盘，记录错因类型。
  3. 建立“连接词+常用句型”积累表。
  4. 每次课后完成 3 个口头造句或书面仿写任务。
  5. 下阶段重点关注阅读细节题和写作句式丰富度。

五、家校沟通话术样例

家长您好，A同学这阶段学习态度比较稳定，课堂能跟上主要内容。现在主要问题不是完全不会，而是阅读题中关键词定位和长句分析还不够稳定，所以有时会在细节题上丢分。接下来我会在课堂中增加“圈关键词、找定位句、比对选项”的训练，也建议孩子课后把错题原因写清楚。我们先以两周为一个小目标，重点提升阅读细节题的稳定性。

六、教师反思

家校反馈要避免空泛评价，例如“孩子不认真”“基础差”。更有效的反馈应包括：

  • 具体表现；
  • 可能原因；
  • 下一步动作；
  • 家庭可配合事项；
  • 下次反馈时间。

这样的沟通更容易获得家长信任，也能帮助学生形成可执行的改进路径。
